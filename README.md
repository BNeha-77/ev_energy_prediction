# Electric Vehicle Energy Demand Prediction Techniques

This project is a web-based platform designed to forecast energy demand for electric vehicles (EVs). It leverages advanced prediction techniques to provide users with accurate energy demand forecasts, helping to optimize energy consumption and improve the efficiency of EV operations.

## Features

- User-friendly interface for inputting data related to electric vehicles.
- Advanced algorithms for predicting energy demand based on historical data and various parameters.
- Integration with Django's admin panel for easy management of data and models.
- Modular architecture allowing for easy expansion and maintenance.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd ev_energy_prediction
   ```

3. Create a virtual environment:
   ```
   python -m venv venv
   ```

4. Activate the virtual environment:
   - On Windows:
     ```
     venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```
     source venv/bin/activate
     ```

5. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the development server:
   ```
   python manage.py runserver
   ```

2. Access the application in your web browser at `http://127.0.0.1:8000/`.
